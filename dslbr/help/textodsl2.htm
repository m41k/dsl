Projeto DSL-BR
 
*O Projeto                                    
  Um simples projeto de tradução colaborativa, 
  todos podem traduzir e/ou usar a tradução.  
                                               
*Mapper's                                      
  Área destinada aos colaboradores, envio de   
  arquivos para tradução do DSL.              
                                              
*Script de Tradução                            
  O script faz download dos arquivos contidos 
  no repositório (arquivos mapper's), e   
  substitui dos arquivos originais no DSL.
